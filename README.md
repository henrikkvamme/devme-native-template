# Devme Native Template

Native iOS and Android starter with a self-hosted Convex backend and Devme.

## Start

Install Devme:

```sh
curl -fsSL https://devme.sh/install | sh
```

_One command is all you need_:

```sh
devme
```

## Customize

Rename the `Starter` targets and replace the `dev.starter.app` identifiers before shipping. Authentication is intentionally omitted.
